# Лабораторная работа №3 - Основы JavaScript

В рамках лабораторной работы были реализованы базовые функции на языке JavaScript, направленные на изучение синтаксиса

# Реализованные функции

1. isInteger(n) — проверка, является ли число целым (с использованием побитовых операторов)
2. even() — возвращает массив чётных чисел от 2 до 20
3. sumTo(n) и recSumTo(n) — сумма чисел до n (итеративно и рекурсивно)
4. factorial(n) — вычисление факториала числа
5. isBinary(n) — проверка, является ли число степенью двойки
6. fibonacci(n) — нахождение n-го числа Фибоначчи
7. getOperationFn(initialValue, operatorFn) — функция-замыкание для выполнения переданной операции
8. sequence(start, step) — генератор арифметической последовательности
9. deepEqual(obj1, obj2) — рекурсивное сравнение объектов по содержимому
